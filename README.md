# Task Manager Flutter App

## Overview
This is a simple Task Manager mobile application built using Flutter.  
It allows users to create, manage, and track daily tasks.

## Features
- Add task with title, description, and date
- Mark task as complete
- Delete task with confirmation
- Persistent local storage (Hive)
- Task list display with date

## Tech Stack
- Flutter (Dart)
- Hive (local storage)
- Material UI

## How to Run
1. Clone repo
2. Run: flutter pub get
3. Run: flutter run
